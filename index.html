Por eso, busco formar parte de un equipo donde pueda crecer profesionalmente, aportar valor con mis conocimientos y, en el futuro, compartir lo aprendido capacitando a otros en este campo.

XIAN Collection

XIAN Collection es una landing page diseñada para ofrecer una experiencia de compra fluida y dinámica. Con un diseño moderno y atractivo, la página incorpora animaciones con JavaScript que aportan dinamismo a la navegación, mientras que el diseño en CSS realza la identidad visual de la marca. Al tratarse de una página única, se implementaron referencias en HTML para redirigir de manera intuitiva a cada sección, asegurando una experiencia de usuario sencilla y eficiente.

Encriptador de texto

Encriptador de Texto es una aplicación web desarrollada como parte del desafío del programa de programación de Alura LATAM. Implementada en JavaScript, permite encriptar y desencriptar mensajes mediante un sistema de reemplazo de palabras, garantizando que el texto transformado pueda ser recuperado fácilmente. La estructura de la página se construyó con HTML, proporcionando la base para los elementos interactivos, mientras que CSS se utilizó para dar estilo a los botones, recuadros y tipografía, logrando una interfaz sencilla y funcional.

Diseño de un blog

Comparador de números

Este proyecto en JavaScript permite al usuario ingresar tres números y los analiza para determinar su orden, mostrando los valores de mayor a menor y de menor a mayor. Además, identifica si los números son iguales y muestra un mensaje correspondiente. Para garantizar una entrada válida, el programa emplea ciclos while que verifican que los datos ingresados sean números. Asimismo, se utilizan funciones específicas para la organización de los valores y la detección de coincidencias. La salida se muestra dinámicamente en el DOM mediante la creación de párrafos y listas, brindando una interfaz clara y estructurada para el usuario.
